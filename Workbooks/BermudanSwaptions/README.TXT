
BERMUDAN SWAPTIONS DEMO
=======================

This demo starts up an Excel session which initializes some market
data and prices an interest rate swap and a bermudan swaption.

To run the demo from a release snapshot or git clone:

- invoke DOS batch file init.bat.

To package up the demo into the following zip file:

    zip/BermudanSwaptions.xip

- invoke DOS batch file make_zip.bat.  You need Python 2.x.
